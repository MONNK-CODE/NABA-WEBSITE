# NABA UIUC Chapter Website

The official website for the **National Association of Black Accountants (NABA)** at the **University of Illinois Urbana-Champaign (UIUC)**. This platform serves as a hub for members, prospective students, and corporate sponsors to engage with the chapter, track events, and learn about the mission of professional excellence.

## Overview

The NABA UIUC website is a custom-built web application designed to showcase the chapter's impact on campus. It provides resources for professional development, networking opportunities in accounting, finance, and consulting, and a community for Black business leaders.

### Key Features

* **Dynamic Events Section:** Real-time updates on upcoming workshops, networking sessions, and general body meetings.
* **Membership Portal:** Information and links for prospective students to join the chapter.
* **Sponsorship Showcase:** A dedicated space for corporate partners to support the chapter’s mission.
* **Executive Board Gallery:** Introductions to the current leadership team with professional links.
* **Photo Gallery:** A visual archive of past events and community building.

## Tech Stack

This project is built using a clean, lightweight front-end stack for maximum performance and easy maintenance:

* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3:** Custom styling with a modern, professional aesthetic.
* **JavaScript (Vanilla):** Interactive components, event handling, and dynamic content rendering.

## 🏗️ Getting Started

To run this project locally:

1. **Clone the repository:**
```bash
git clone https://github.com/MONNK-CODE/NABA-WEBSITE.git

```

2. **Navigate to the directory:**
```bash
cd NABA-WEBSITE

```


3. **Open the site:**
   Simply open `index.html` in your preferred web browser.
